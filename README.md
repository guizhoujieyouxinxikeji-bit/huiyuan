# 餐饮门店会员小程序

> 原生微信小程序 · 高端商务风格（金黑主题）

面向餐饮门店的会员管理系统小程序前端，支持会员储值充值、老客推荐奖励、积分管理等功能。

## 功能概览

### 用户端
- 会员首页：等级展示、积分余额、快捷操作
- 会员码：二维码扫码识别
- 储值充值：预设套餐 + 自定义金额，充值赠送10%积分
- 积分明细：获得/消费/过期分类查看
- 积分商城：积分兑换餐品酒水
- 推荐有礼：一级推荐，推荐充值/消费奖励10%
- 消费记录：历史消费和积分抵扣明细
- 个人中心：会员信息、功能入口

### 管理后台
- 数据看板：今日概览 + 累计统计
- 会员管理：搜索、详情查看
- 推荐关系：一级推荐关系对
- 充值/消费记录：筛选、数据导出
- 积分流水：6种类型分类
- 积分核销：扫码核销（店员操作）
- 系统设置：奖励比例、风控参数

## 核心业务规则

| 规则 | 说明 |
|------|------|
| 充值赠送 | 充值金额 x 10% = 赠送积分 |
| 推荐充值奖励 | 被推荐人充值时，推荐人获得充值金额 x 10% 积分 |
| 推荐消费奖励 | 被推荐人消费时，推荐人获得消费金额 x 10% 积分 |
| 积分抵扣 | 1积分 = 1元，仅限店内消费 |
| 推荐层级 | 仅一级推荐 |
| 月上限 | 推荐人每月奖励上限5000积分 |
| 有效期 | 积分365天有效 |
| 退款扣回 | 退款/撤单自动扣回积分 |

## 快速开始

1. 安装微信开发者工具
2. 导入项目，填入你的小程序 AppID
3. 修改 app.js 中 baseUrl 为实际后端地址
4. 编译预览

## 部署注意

- TabBar 图标需自行准备6张PNG放入 miniprogram/images/
- 充值功能需对接微信支付
- 后端接口参照 docs/API.md 实现
